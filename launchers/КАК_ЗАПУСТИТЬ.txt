X-Rav для Windows
==================

Структура архива:
  x64/                       — собранное приложение для 64-битной Windows (X-Rav.exe + .dll)
  x86/                       — то же для 32-битной Windows
  Setup-Defender.bat         — добавляет исключения Defender (запусти ОДИН раз)
  Запустить-x64.bat          — запуск 64-битной версии (с UAC + защита от MOTW)
  Запустить-x86.bat          — запуск 32-битной версии
  КАК_ЗАПУСТИТЬ.txt          — этот файл
  README.md / LICENSE


ПОРЯДОК ДЕЙСТВИЙ (важно!)
=========================

ШАГ 1) ПКМ по «Setup-Defender.bat» → «Запуск от имени администратора».
       Согласись на UAC. Скрипт добавит в исключения Microsoft Defender:
         - папку с приложением,
         - %APPDATA%\X-Rav,
         - %LOCALAPPDATA%\Temp\.net,
         - процессы X-Rav.exe / xray.exe / sing-box.exe / hev-socks5-tunnel.exe.
       Это делается ОДИН раз. Без этого Defender эвристически удаляет xray.exe
       при первой распаковке (он не подписан сертификатом — false-positive).

ШАГ 2) Если ты до этого уже пытался запустить и Defender что-то удалил —
       РАСПАКУЙ АРХИВ ЗАНОВО (поверх старой папки). Иначе .exe или его
       зависимости отсутствуют.

ШАГ 3) Двойной клик по «Запустить-x64.bat» (или «Запустить-x86.bat» для
       32-битной системы). Скрипт сам:
         - запросит UAC (Wintun без админа не работает),
         - снимет Mark-of-the-Web (MOTW) с файлов в папке,
         - запустит X-Rav.
       Если приложение упадёт — окно НЕ закроется, покажет код ошибки
       и подсказку.


ИМПОРТ КЛЮЧЕЙ
=============
Вкладка «Серверы»:
- вставь vless://, vmess://, trojan://, ss://, hysteria2:// или tuic://;
- «Из буфера» — вставит то, что скопировано;
- «Подписка» → URL подписки → «Добавить».

Кнопка «Пинг» возле каждого ключа измерит TCP RTT до сервера.
«Пинг всех» — проверит все ключи разом.

Бэкенды (выбираются автоматически):
  VLESS / VMess / Trojan / SS / JSON  → xray + hev-socks5-tunnel (системный TUN)
  Hysteria2 / TUIC                    → sing-box (свой TUN, hev не нужен)


АВТО-ВЫБОР (Smart Mode)
=======================
В Настройках → «Режим» → выбери «Авто». Приложение само пингует все ключи
раз в 30 секунд, выбирает лучший по latency и переключается на следующий
при сбое подключения. На главном экране в авто-режиме — только power-кнопка
и строка «Подключено к: <name> · <ms> мс».


ТЕМА И ЯЗЫК
===========
Настройки → «Внешний вид»: тема (светлая / тёмная / системная) + язык
(Русский / English / Türkçe). Применяется без перезапуска.


URL-СХЕМА x-rav://
==================
Настройки → «URL-схема» → «Зарегистрировать». После этого ссылки вида:
  x-rav://import?url=vless%3A%2F%2F...
  x-rav://sub?url=https%3A%2F%2F...
будут открывать приложение и импортировать ключ/подписку.


ЕСЛИ ЧТО-ТО НЕ ТАК
==================
1) Окно вообще не появляется и при двойном клике .exe «ничего не происходит»:
   → Defender съел .exe. Запусти Setup-Defender.bat → распакуй архив заново.

2) «Запустить-x64.bat» открывается и сразу закрывается:
   → Старая версия .bat (без pause-on-error). Скачай свежий архив.

3) В логе «xray сразу завершился (код -1)» БЕЗ stderr:
   → Defender удалил xray.exe из %APPDATA%\X-Rav\tools. Запусти
     Setup-Defender.bat и перезапусти приложение.

4) SmartScreen «Windows защитила ваш компьютер»:
   → «Подробнее» → «Выполнить в любом случае».
